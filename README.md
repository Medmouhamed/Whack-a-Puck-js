# 🎯 Whack-a-Puck: Neon Edition

A high-energy, interactive web game built with Vanilla JavaScript and modern CSS3. Test your reflexes by clicking on the glowing "pucks" before they disappear in this Cyberpunk-themed challenge.

## 🎮 Game Overview
**Whack-a-Puck** is a customizable reaction game. The objective is to click on the randomly appearing active squares to score as many points as possible within the time limit.

## 🚀 Key Features
* **Fully Customizable**: Players can adjust the game duration, board size (rows/cols), and the speed of the puck movement directly from the UI.
* **Dynamic Grid Generation**: The game board is built dynamically using JavaScript based on user input.
* **Immersive UI**: Features a futuristic "Cyberpunk" aesthetic with:
    * Radial and conic gradients.
    * Glassmorphism (backdrop-blur effects).
    * Neon glowing animations and hover transforms.
* **Responsive Logic**: Smooth gameplay with real-time score tracking and a countdown timer.

## 🛠️ Technologies Used
* **HTML5**: Semantic structure for the game interface.
* **CSS3 (Advanced)**: Utilizing CSS animations, `@keyframes`, custom properties, and Grid Layout.
* **JavaScript (ES6)**: 
    * DOM manipulation for dynamic board creation.
    * Interval management for timing and movement logic.
    * Event handling for interactive gameplay.
